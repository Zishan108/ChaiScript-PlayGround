☕ ChaiScript Docs

A chai-flavored interpreted programming language.

ChaiScript is an interpreted language best suited for learning coding. It is compatible with every operating system and runs inside your browser via the ChaiScript Playground.

The language has a similar syntax and flow to Python, JavaScript and shell programming — it is highly inspired by Python — with a twist of chai-oriented keywords that make it fun to write.

Like Python, you can code in ChaiScript by both interactive mode (directly through command line) and Script mode (by making a .chai extension file and running it through the command line).

Getting Started

The easiest way to use ChaiScript is the online playground — no installation needed. Just open the editor and start writing code.

▶ Open Playground

You can also download the ChaiScript Interpreter and run .chai files locally. A VS Code extension is available for syntax highlighting and autocomplete.

Basic Syntax

ChaiScript uses new lines to complete commands. You can also use semicolons ; (paying tribute to Java).

There is no use of indentation or curly braces for separating code blocks — blocks are opened with then_chai and closed with sip.

Hello World

serveChai("Hello, World!")

Variables

Variables are containers for storing data. Use the cup keyword to create one.

ChaiScript is a dynamically typed language — no need to declare a type. A variable is created when you first assign a value to it, and can change type at any time.

cup a = 12 cup name = "Zishan" cup flag = true

Case Sensitive

Variable names are case-sensitive. cup x = 5 and cup X = "hello" are two completely separate variables.

Type Casting

cup x = int("42") cup y = str(100) cup z = float("3.14")

Get the Type

serveChai(type_chai(42)) # Output: Number serveChai(type_chai("hi")) # Output: String

Data Types

ChaiScript has the following built-in data types:

Example Type
cup x = 42 Number (int)
cup y = 3.14 Number (float)
cup s = "Alice" String
cup l = [1, 2, 3] List
cup d = {"key": "val"} Dictionary
cup b = true Boolean

Numbers

There are two numeric types: int (whole numbers) and float (decimals).

Int

cup x = 1 cup y = 35656222554887711 cup z = -3255522 serveChai(type_chai(x)) # Number

Float

cup x = 1.10 cup y = 1.0 cup z = -35.59 serveChai(type_chai(x)) # Number

Strings

Strings in ChaiScript are surrounded by double quotation marks.

serveChai("Hello") serveChai("World!")

Assign to a Variable

cup a = "Zishan" serveChai(a)

String Length

cup a = "ChaiScript" serveChai(str_len(a)) # 10 serveChai(len(a)) # 10

Lists

Lists store multiple items in a single variable. Created using square brackets.

cup my_list = ["apple", "banana", "cherry"] serveChai(my_list) # Output: ["apple", "banana", "cherry"]

List items are ordered, changeable, and allow duplicates. Items can be of any data type.

List Length

cup my_list = ["apple", "banana", "cherry"] serveChai(list_len(my_list)) # 3 serveChai(len(my_list)) # 3

Dictionaries v1.1.0

Dictionaries store key–value pairs, letting you organize data and fetch values quickly by their keys.

cup student = {"name": "Zishan", "age": 20, "course": "IT"} serveChai(dict_get(student, "name")) # Output: Zishan serveChai(dict_get(student, "age")) # Output: 20

Booleans

Only true and false are Boolean types (not 0 and 1).

cup isActive = true cup isComplete = false serveChai(type_chai(isActive)) # Number (true = 1 internally)

Conditionals

ChaiScript uses hot (if), mild (else if), and chilled (else). Blocks open with then_chai and close with sip.

Syntax

hot condition then_chai # code mild other_condition then_chai # code chilled # code sip

Example

cup a = 12 cup b = 1212 hot b > a then_chai serveChai("b is greater than a") mild b == a then_chai serveChai("b is equal to a") chilled serveChai("a is greater than b") sip
Note: sip is written only once for the entire hot/mild/chilled block — it closes the whole conditional scope.

and / or / not examples

cup a = 19 hot a > 18 and a < 100 then_chai serveChai("Valid age!") sip hot a> 18 or a <= 19 then_chai serveChai("Passes at least one check") sip hot not a < 18 then_chai serveChai("Not underage") sip

Functions

Functions are defined using the brew keyword and closed with sip.

Declaration

brew greet(name) serveChai("Hello " + name) sip greet("Zishan") # Output: Hello Zishan

One-line Function (Arrow syntax)

brew square(x) -> x * x serveChai(square(5)) # Output: 25
Note: One-line functions using arrow syntax do not need sip.

Return Statement

Use return_chai to return a value.

brew add(a, b) return_chai a + b sip cup result = add(3, 4) serveChai(result) # Output: 7

For Loop

Used to iterate a code block a specific number of times. Keyword: for_chai.

Syntax

for_chai i = start to_chai end then_chai # code block sip

Example

for_chai i = 0 to_chai 7 then_chai serveChai(i) sip
Output
0
1
2
3
4
5
6
Note: to_chai is exclusive — the loop runs up to but does not include the end value.

Using STEP

for_chai i = 5 to_chai 0 STEP -1 then_chai serveChai(i) sip
Output
5
4
3
2
1

While Loop

Executes a block as long as a condition is true. Keyword: while_chai.

while_chai condition then_chai # code block sip

Example

cup a = 1 while_chai a <= 5 then_chai serveChai(a) cup a=a + 1 sip
Output
1
2
3
4
5

Break & Continue

break_chai exits a loop early. continue_chai skips the current iteration.

Break Example

for_chai i = 0 to_chai 5 then_chai hot i == 3 then_chai break_chai sip serveChai(i) sip
Output
0
1
2

Continue Example

for_chai i = 0 to_chai 5 then_chai hot i == 3 then_chai continue_chai sip serveChai(i) sip
Output
0
1
2
4

Exception Handling v1.1.0

Use try_chai / catch_chai to handle runtime errors gracefully.

Basic Structure

try_chai then_chai # code that may throw an error catch_chai then_chai # handle the error here sip

Division By Zero

try_chai then_chai cup x = 5 / 0 catch_chai then_chai serveChai("Error: Division By Zero") sip

Capture the Error Object

try_chai then_chai cup x = 10 / 0 catch_chai e then_chai serveChai("Caught: " + e) sip
Best Practices:
• Use try_chai to wrap code that might fail.
• Keep try blocks small — only wrap the risky statements.
• sip always closes the try/catch block.

File Handling v1.2.0

Read, write, and append files using chai-inspired file functions.

Function Description
brew_file(path, mode) Opens a file. Mode: "r", "w", "a"
pour_file(handle, text) Writes text to the file
sip_file(handle) Reads the file content
seal_file(handle) Closes the file

Write to a file

cup f = brew_file("test.txt", "w") pour_file(f, "Hello from ChaiScript!") seal_file(f)
Important: Always pass the file handle returned by brew_file to pour_file, sip_file, and seal_file — not the file name string.

Built-in Functions — Utility

serveChai(arg)

Prints the argument to the screen.

serveChai("Hello, Chai!") # Output: Hello, Chai!

serve_ret(arg)

Prints and returns the argument.

cup x = serve_ret("Hi") # Prints: Hi, assigns "Hi" to x

orderChai(prompt)

Prompts for string input.

cup name = orderChai("Enter name: ")

orderChai_int(prompt)

Prompts for integer input.

cup age = orderChai_int("Enter age: ")

clean() / cls()

Clears the terminal.

type_chai(arg)

Returns the type of the argument.

serveChai(type_chai(42)) # Output: Number

current_date_time()

Returns the current date and time as a string.

serveChai(current_date_time()) # e.g., "2025-02-21 10:00:00"

int(arg) / str(arg) / float(arg)

Convert between types.

cup n = int("123") # 123 cup s = str(42) # "42" cup f = float("3.14") # 3.14

time_freeze(seconds)

Pauses execution for the specified number of seconds.

serveChai("Waiting...") time_freeze(2) serveChai("Done!")
